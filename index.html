Otter Framework API (1.0)

Download OpenAPI specification:Download

License: Apache 2.0

The Otter Framework REST API is for API consumers creating and manipulating room resources for peer to peer video communications.

Errors

The API uses standard HTTP status codes to indicate the success or failure of the API call. The body of the response will be JSON in the following format:

{
  "message": "resource not found"
}

Create a new room

Request Body schema: application/json

(Optional) Specify properties for the new room. If there is no request body, a new room with default settings will be created.

room_name
string

Responses

http://localhost:3000/v1/rooms

Fetch a Room resource

Fetch a specific Room resource by a unique room identifier (can be room_id or room_name).

path Parameters
uniqueIdentifier
required
string

Responses

http://localhost:3000/v1/rooms/{uniqueIdentifier}
